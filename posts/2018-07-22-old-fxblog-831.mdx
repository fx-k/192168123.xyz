下面分享一段可以用的（不排除以后被x）：

更新：2018 .10.13最新屏蔽悬浮窗的JS：
```js
//CHANGYAN-gun!
window.onload=function (){
var div = document.getElementById("pop_ad");
div.style.display = "none";
}
```
CSS:
```css
#feedAv{ margin-top: -250px!important;transform: scale(0);}
```
将上面这条css加入到你的主题style.css里面，刷新缓存，即可！
